# 📚 Estudos de API com Spring Boot

Este repositório foi criado com o objetivo de documentar minha jornada de aprendizado no desenvolvimento de APIs utilizando **Spring Boot**. Aqui você encontrará exemplos práticos, testes, anotações em código e experimentações baseadas no que estou estudando.

---

## 🚀 Objetivo

Consolidar conhecimentos em desenvolvimento backend, com foco em:

- Criação de APIs RESTful  
- Boas práticas de desenvolvimento  
- Organização de projetos com Spring Boot  
- Integração com bancos de dados  
- Testes e validações  

---

## 🛠️ Tecnologias Utilizadas

- Java  
- Spring Boot  
- Spring Web  
- Spring Data JPA  
- Maven/Gradle
- Docker
- Banco de dados (H2, MySQL ou outro conforme evolução dos estudos)  

---

## 📖 Sobre os Estudos

Este repositório não segue um padrão fixo de projeto, pois seu propósito é educacional. Aqui você encontrará:

- Testes de conceitos  
- Implementações simples  
- Refatorações ao longo do tempo  
- Código em evolução constante  

---

## ⚠️ Observações

- O código pode não estar otimizado ou finalizado  
- Algumas implementações podem ser apenas experimentais  
- Estruturas podem mudar conforme avanço nos estudos  

---

## 💡 Motivação

Este repositório serve como um diário de aprendizado e evolução como desenvolvedor backend. A ideia é aprender fazendo, errando e melhorando continuamente.
